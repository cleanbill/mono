Setup tutorial, Kickoff tutorial, Whistle tutorial.

0. Aim

This article steps through how to create a football score card for parents who wanna keep a track of their kids footy game. This will be consist of three screens: Setup and Kick off and whistle blown screen.


1. Setup

Setup screen will include:
  • Home / Away toggle
  • Edit both teams names
  • Kick off buttons
  • Input players name and position
    (including goal and bench)
  • Publish players formation button
  • A list of events with undo and share buttons
Go to building setup screen page

2. Kick Off

Kick off screen will include:
  • Both teams names with scores
    - opposition name button for concede a goal
  • Clock
  • whistle blown button
  • Players name and position scoring buttons
    (including goal and bench)
  • A list of events with undo and share buttons

3. Whistle Blown

Whistle Blown screen will include:
  • Home / Away toggle
  • Final Whistle
  • Both teams names (editable) and score
  • Input players name and position
    (including goal and bench)
  • Rest clock Button (Half time)
  • Play on Button
  • Publish players formation button
  • A list of events with undo and share buttons
